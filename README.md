# Hospital Analytics Dashboard (Power BI)

## Project Overview

This project demonstrates how hospital operational data can be transformed into actionable insights using Power BI. The dashboard analyzes patient admissions, disease trends, insurance coverage, and hospital service utilization to help healthcare administrators make data-driven decisions.

The goal of this project is to simulate a real-world healthcare analytics environment where hospital executives can monitor key performance indicators (KPIs) and identify trends in patient care.

---

## Business Questions Answered

* How many patients are admitted each month?
* What are the most common diseases diagnosed in the hospital?
* What percentage of patients are covered by HMO or PhilHealth?
* What hospital services are most utilized?
* What age groups represent the largest patient population?

---

## Dashboard Features

* Total Inpatient and Outpatient KPIs
* HMO vs PhilHealth Patient Analysis
* Age Group Patient Distribution
* Top 10 Diseases based on Final Diagnosis
* Monthly Admission Trends
* Service Utilization Analysis

---

## Data Model

The dashboard is built using multiple hospital data tables including:

* Patient Information
* Admission Records
* Service Transactions
* Insurance Data
* Final Diagnosis Records

These tables were modeled to support efficient filtering and reporting across the dashboard.
